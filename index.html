  1. Supervised Learning - The model is trained with numerous samples of labeled data. In other words, the training data is complete with the expected output given specific inputs. The model can then use this information to predict the output for unlabeled data.

  2. Unsupervised Learning - In contrast to supervised learning, the model is trained with unlabeled data. This data would simply be a number of inputs. The model is then tasked with finding similarities in the data and creating its own outputs instead of using predefined outputs as with supervised learning.

  3. Reinforcement Learning - Reinforcement learning can be related to a trial-and-error approach. An agent is placed in an environment. The agent will be rewarded each time it improves toward the desired behavior/output. The model aims to maximize this reward, so it will disregard trials that recieve little or no reward.
